# Python-Basics

一个简单的 Python 基础练习代码仓库，包含 Python 基础语法和常用库的学习示例。

## 目录结构

- `Python基础语法/` - Python 基础语法练习
  - 列表、字典、元组等数据结构
  - 函数、文件操作
  - 日期计算、文件查找工具
  - 人生重开模拟器等趣味小程序

- `库的使用/` - Python 常用库应用示例
  - Excel 操作（openpyxl）
  - 二维码生成（qrcode）
  - 学生管理系统（可打包成 exe）
